GUI Parameter Details

GENFIRE Main Window

"Select Projection File" - Choose filename containing projection images. Alternatively you enter a file by editing the text, but it must be a valid file type.
"Use default support" - Automatically generate a cubic support defined by the oversampling ratio. For example if the projections are size 128x128 and the oversampling ratio is set to 3, the default support will be 384x384x384 with a central 128x128x128 region of 1's and the remaining region set to 0. Uncheck to enter a filename manually.
"Select Support File" - Choose filename containing the 3D support. Alternatively you enter a file by editing the text, but it must be a valid file type. The array width should match that of the inputted projections.
"Select Angle File" - Choose filename with the Euler angles. "Provide initial object" - Check to enable selection of an intial model. "Select Initial Object File" - Choose filename containing the initial model. "Results Filename" - Filename for the output volume. Must be a valid filetype. Error curves are saved as .txt files with the same base filename as the results. "Resolution Extension/Suppression" - Choose the style of constraint enforcement.

  • On - Enforce only lowest resolution information on initial iterations, extending slowly until all information is enforced by the middle iterations, and then reverse the process. This is more robust to noise.
  • Off - Enforce all information at all times.
  • Extension Only - Enforce only lowest resolution information on initial iterations, extending slowly until all information is enforced by the final iterations.

"Number of Iterations" - Number of GENFIRE iterations to use
"Oversampling Ratio" - Final oversampling ratio to use when gridding the projections. Larger oversampling ratios will produce more accurate griddings at the cost of larger array size. Ideally use at least 3.
"Interpolation Cutoff Distance (pixels)" - Size of the interpolation kernel used for gridding. Small radii will result in an assembled grid with fewer values, but more accurate ones. Conversely, larger values for this parameter will be more accepting of datapoints, but at the cost of accuracy. The default 0.7 was found to be ideal based on simulations.
"Summarize Results" - Display a summary of reconstruction results. Choose a volume file, and projections and central slices along the three principal directions will appear. Error curves will be plotted if the corresponding files exist. "Launch Reconstruction" - Begin the reconstruction, if the parameters are valid.

Projection Calculator

"Model" - Filename of volume from which to calculate projections.
"Clear Model" - Clear display and model from memory.
"Calculate Projection Series from Model" - Use the current model to simulate a series of projections.
"Phi" - set Phi Euler angle interactively.
"Theta" - set Theta Euler angle interactively.
"Psi" - set Psi Euler angle interactively.

Volume Slicer

"X" - Set X value for the YZ plane.
"Y" - Set Y value for the XZ plane.
"Z" - Set Z value for the XY plane.
"Lock Colormap" - If checked, locks the colormap for each image based on the current ones. If you want to lock a different colormap, uncheck the box, move the slicer, and recheck the box to lock the new setting.
